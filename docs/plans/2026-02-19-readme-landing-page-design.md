# README ランディングページ設計書

## 概要

README.md を施術者向けマーケティングランディングページとして再構成する。

## ターゲット

整骨院・鍼灸院・マッサージ院等の小規模施術者本人。技術に詳しくない。

## 構成（ストーリー型）

### 1. ヒーロー
- プロダクト名「semr」+ キャッチコピー
- 1行サブコピーで何ができるかを伝える
- CTA: https://semr.nouer.net

### 2. 課題提起 → 解決
- 小規模施術院の電子カルテの課題（高い、複雑、大げさ）
- semr の解決策（サーバー不要、無料、すぐ使える）

### 3. 主な機能（スクリーンショット付き）
- 患者一覧（`docs/images/04_patients_list.png`）
- SOAP記録 + バイタル（`docs/images/08_karte_saved.png`）
- 診療履歴タイムライン（`docs/images/14_history.png`）
- 他: 処方管理、検査結果、AI診断支援

### 4. 特徴（差別化ポイント）
- 完全オフライン対応（PWA）
- データは端末から外に出ない
- JSONバックアップ
- スマホ・タブレット対応

### 5. CTA（クロージング）
- デモリンク再掲

## 言語
日本語のみ

## スクリーンショット
2-3枚（患者一覧、カルテ記録、診療履歴）
