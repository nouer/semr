# semr — シンプル電子カルテ

**紙カルテを卒業。ブラウザだけで、今日から使える電子カルテ。**

整骨院・鍼灸院・マッサージ院・個人クリニック向けに設計された、完全無料のブラウザ電子カルテシステムです。
サーバー不要、インストール不要。ブラウザを開くだけで使い始められます。

**[今すぐ試す →](https://semr.nouer.net)**

---

## なぜ semr？

小規模な施術院にとって、電子カルテは「高すぎる・複雑すぎる・大げさすぎる」。
semr は、紙カルテの手軽さとデジタルの便利さを両立するために作られました。

- **サーバー不要** — データはお使いの端末内（ブラウザ）に保存。外部に送信されません
- **月額 0 円** — ずっと無料で使えます
- **3 分で開始** — アカウント登録不要。ブラウザでアクセスすれば即利用開始

## 主な機能

### 患者管理

患者情報の登録・検索・編集をシンプルな画面で。
患者コードの自動採番、ふりがな検索、アレルギー・既往歴の管理にも対応しています。

![患者一覧](docs/images/04_patients_list.png)

### SOAP カルテ記録

医療現場で広く使われる SOAP 形式（主訴・所見・評価・計画）で診療記録を作成。
バイタルサイン（体温・血圧・脈拍・SpO2 等）の記録や、写真添付にも対応しています。

![カルテ記録](docs/images/08_karte_saved.png)

### 診療履歴タイムライン

診療記録・処方・検査結果を時系列で一覧表示。
患者ごとの診療の流れをひと目で把握できます。

![診療履歴](docs/images/14_history.png)

### その他の機能

- **処方管理** — 薬剤名・用量・用法・日数を記録
- **検査結果管理** — 検査データの入力と一覧表示、正常/異常の自動判定
- **バイタル推移グラフ** — 体温・血圧などの推移をグラフで可視化
- **AI 診断支援** — OpenAI API 連携による診断補助（オプション）
- **データエクスポート/インポート** — JSON 形式でバックアップ・復元・他端末への移行

## サンプルデータ

デモや動作確認用に、リアルなサンプルデータを用意しています。

**[sample_data.json をダウンロード](sample_data.json)**

| 項目 | 件数 |
|------|------|
| 患者 | 100人 |
| カルテ（SOAP 診療記録） | 10,000件 |
| 処方 | 10,000件 |
| 検査結果 | 10,000件 |

アプリの「データ」タブからインポートするだけで、すぐにデモ環境を構築できます。

<details>
<summary>含まれるデータの詳細</summary>

- **患者**: 5〜92歳の幅広い年齢層、男女均等、アレルギー情報・既往歴・緊急連絡先付き
- **カルテ**: 35種類の SOAP テンプレート（咳・発熱・めまい・腰痛・高血圧など）、バイタルサイン付き
- **処方**: 41種類の実在薬剤名（NSAIDs・抗菌薬・降圧薬・糖尿病薬・漢方・外用薬等）、用法用量・日数付き
- **検査**: 血液検査・尿検査・画像検査・その他、基準値と自動判定（正常/要注意/異常）付き

</details>

サンプルデータの再生成:

```bash
node tools/generate_sample_data.js
```

## semr の特徴

| | |
|---|---|
| **完全オフライン対応** | インターネット接続なしで動作。PWA としてホーム画面に追加可能 |
| **プライバシー保護** | 患者データは端末内の IndexedDB に保存。サーバーへの送信は一切なし |
| **データポータビリティ** | JSON エクスポートでバックアップ。端末の移行もファイル 1 つで完了 |
| **マルチデバイス** | PC・タブレット・スマートフォンに対応したレスポンシブデザイン |

## 開発者向け情報

### 技術スタック

| 項目 | 技術 |
|------|------|
| フロントエンド | HTML + vanilla JavaScript (SPA) |
| データストア | IndexedDB |
| グラフ描画 | [Chart.js](https://www.chartjs.org/) v4 |
| PWA | Web App Manifest + Service Worker |
| コンテナ | Docker (nginx:alpine / node:alpine) |
| デプロイ | Vercel |

### セットアップ（ローカル開発）

```bash
# Docker でビルド＆起動
bash scripts/build.sh

# ブラウザで http://localhost:8083 にアクセス
```

### テスト

```bash
docker compose run --rm emr-test npm test
```
